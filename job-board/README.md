# GraphQL Job Board

Sample application used in the [GraphQL by Example](https://www.udemy.com/course/graphql-by-example/?referralCode=7ACEB04674F000BAC061) course.

## Steps to start-up

### Server project
#### Install dependencies
npm install

#### Start Nodemon server
npm start

### Client project
#### Install dependencies
npm install

#### Start React server
npm start
